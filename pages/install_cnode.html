Installation Cnode


Requirements

Installation of the Central node was tested under Ubuntu version 20.04. The Memory of a computer/virtual machine for installation the Central node must be not less than 8GB (16GB and high recommended), Hard disk not less than 40GB. Also, the computer should be reachable for remote components of the solution via hostname (see file /etc/hosts or DNS service). Next TCP/UDP ports should be open for interaction between solution components and users:

Installation

  • Download installation files

git clone https://github.com/alertflex/cnode.git
cd cnode

  • Fill in project-specific parameters in file env.sh, as an example you can use file example_env.sh

  • Start installation

chmod u+x install_ubuntu20.sh
./install_ubuntu20.sh

  • After finish a software installation, reboot the system
