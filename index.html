Bridging the gap between web services monitoring and testing

Alert triage: normalization, prioritization, and confirmation

Detection of bugs, threats, and vulnerabilities

Integrated analysis of containers, hosts, and networks

DevSecOps orchestration

Alertflex acts as an orchestrator for a distributed grid of DAST/TAF scanners and HIDS/NIDS sensors. It helps software teams quickly implement DAST in CD/GitOps processes, enables analytics and observability of detected information using OpenSearch.

If you have a question or need tech support, or consulting, please send an email to the contact address info@alertflex.org

Alertflex features

  • Testing and monitoring a complex web services infrastructure, not just the web application itself

  • A unified report for multiple remote scans, including alerts summary from scanners and sensors, serving as a quality gate

  • Several correlation methods for scan findings, along with rules for logging and dismissing alerts

  • Using IDS sensors to monitor system events and detect hidden bugs

  • Selective logging and inspecting HTTP traffic

  • Converting IDS events to OCSF format, suited for the security analytics and threat intelligence functions of OpenSearch

  • Exporting alerts to external issue trackers and unified posture reports to external CI/CD tools

  • Asynchronous, non-blocking scans triggered by API calls in external CI/CD pipelines

  • Seamless connectivity between central management and remote collectors/runners
